# 🏠 Kost Management System

A web-based Kost Management System developed using PHP and MySQL to streamline boarding house operations, including room management, resident data, payment tracking, and maintenance handling.

## 🚀 Features

### 👤 Authentication
- User registration & login system
- Session-based authentication
- Role separation (Admin & Resident)

### 🛏️ Room Management
- Add, edit, and delete rooms
- Track room availability and details

### 🧑‍🤝‍🧑 Resident Management
- Manage resident data
- Edit and update resident information

### 💰 Payment Tracking
- Record and monitor payments
- Update payment status

### 🛠️ Maintenance Requests
- Residents can submit maintenance requests
- Admin can review and update request status

### 📊 Admin Dashboard
- Centralized control panel
- Overview of rooms, residents, payments, and maintenance

## 🧰 Tech Stack
- PHP (Native)
- MySQL
- HTML, CSS

## 📁 Project Structure
/admin → Admin panel (dashboard & management)
/resident → Resident dashboard
/config.php → Database configuration
/login.php → Authentication system
/register.php → User registration


## ⚙️ Setup Instructions
1. Clone this repository
2. Import `kost_management.sql` into your MySQL database
3. Configure database in `config.php`
4. Run using local server (XAMPP / Laragon)
